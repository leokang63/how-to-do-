About

사이트 소개

Monster Memo Hunt는 사용자가 해야 할 일을 조금 더 재미있고 직관적으로 관리할 수 있도록 만든 정적 Todo 리스트 서비스입니다. 단순히 체크박스를 채우는 방식에서 벗어나, 각각의 일을 하나의 몬스터로 설정하고 정해진 시간에 맞춰 토벌하는 흐름을 제공합니다.

이 사이트는 HTML, CSS, JavaScript만으로 동작하도록 제작되어 Cloudflare Pages 같은 정적 호스팅 환경에 쉽게 배포할 수 있습니다. 서버 계정 생성이나 로그인 없이 사용할 수 있으며, 작성한 데이터는 사용자의 브라우저 저장소에 보관됩니다.

종이 위에 손글씨로 적는 듯한 시각적 분위기를 목표로 했으며, 한국어와 영어가 모두 자연스럽게 보이도록 웹폰트와 시스템 폰트 대체 설정을 함께 적용했습니다.

Features

기능 소개

몬스터형 Todo 등록

해야 할 일, 몬스터 이름, 난이도, 날짜와 시간을 입력해 하나의 토벌 목표로 관리할 수 있습니다.

브라우저 자동 저장

등록한 내용은 localStorage에 저장되므로 같은 기기와 브라우저에서 다시 접속해도 목록이 유지됩니다.

상태 필터

전체, 대기 중, 토벌 완료 상태로 목록을 빠르게 분류해 현재 해야 할 일을 확인할 수 있습니다.

정적 배포 최적화

별도 백엔드 없이 동작하도록 구성했으며, SEO 메타 태그와 구조화 데이터를 포함했습니다.

Privacy policy

개인정보처리방침

Monster Memo Hunt는 기본 사용 과정에서 회원가입, 로그인, 이름, 이메일 주소, 전화번호와 같은 개인식별정보를 요구하지 않습니다. 사용자가 입력한 할 일, 몬스터 이름, 메모, 날짜 및 시간 정보는 서버로 전송되지 않고 사용자의 브라우저 localStorage에 저장됩니다.

브라우저 저장소에 보관된 데이터는 사용자가 이용 중인 기기와 브라우저 환경에 남아 있으며, 브라우저 데이터 삭제, 시크릿 모드 종료, 저장소 초기화 등의 경우 사라질 수 있습니다. 중요한 업무 기록은 별도로 백업하는 것을 권장합니다.

향후 광고 서비스, 웹 분석 도구, 문의 기능 등 외부 서비스가 추가될 경우 쿠키, 광고 식별자, 접속 통계와 관련된 처리 항목이 발생할 수 있습니다. 이 경우 실제 적용 전에 개인정보처리방침을 갱신하고, 필요한 고지와 동의 절차를 제공할 예정입니다.

개인정보 관련 문의가 필요한 경우 사이트 운영자가 별도로 공개한 연락 수단을 통해 문의할 수 있습니다. 본 문구는 정적 사이트 운영을 위한 기본 안내이며, 실제 서비스 공개 전 운영 주체, 연락처, 광고 및 분석 도구 사용 여부에 맞춰 최종 검토해야 합니다.

FAQ

자주 묻는 질문

회원가입이 필요한가요?

아니요. Monster Memo Hunt는 로그인 없이 사용할 수 있는 정적 Todo 웹앱입니다.

작성한 내용은 어디에 저장되나요?

할 일과 메모는 사용자의 브라우저 localStorage에 저장되며, 서버로 전송되지 않습니다.

다른 기기에서도 볼 수 있나요?

현재 버전은 같은 기기와 같은 브라우저에서만 저장 내용을 다시 볼 수 있습니다.

Cloudflare Pages에 적합한가요?

HTML, CSS, JavaScript만으로 구성되어 정적 호스팅 환경에 바로 배포할 수 있습니다.
